# CJ올리브영 백엔드 포지션 심층 리서치 (2026.07.21)

CJ올리브네트웍스 CJ ONE과 나란히 비교 검토용. **주의**: CJ올리브영 ≠ CJ올리브네트웍스(별도 법인) — 잡플래닛에도 두 개 회사코드(23194 구법인, 442105 신법인/2023년 물적분할 이후)가 따로 존재하니 리뷰 볼 때 헷갈리지 말 것.

## CJ올리브영 조직 평판

**주의**: 잡플래닛 상세 페이지는 403으로 직접 접근 실패(CJ올리브네트웍스 리서치 때와 동일한 제약), 검색 스니펫·잡코리아·블라인드·캐치 기준.

- **잡코리아**: 종합 4.1/5.0. 세부(100점 만점 환산 추정): 급여 35, 워라밸 44, 조직문화 51, 복리후생 59, 고용안정성 66, **커리어성장 74** — 커리어 성장 항목이 가장 높게 나옴
- **캐치**: 재직자 추천율 58%
- **잡플래닛**: 구법인(23194) 리뷰 5,908건·면접후기 1,821건(면접난이도 2.7), 신법인(442105, 2023년 물적분할 이후) 리뷰 309건·면접후기 7건(면접난이도 3.0) — 지원 시 어느 법인 채용인지 확인 권장(대부분 신법인 CJ올리브영 기준일 가능성 높음)
- **블라인드**: "워라밸 극악" 표현이 반복 언급되나 **전략 시즌(성수기) 한정**이라는 맥락 — 개발직군 후기는 "재택 팀바팀이지만 주 2~3일, 많게는 5일도 함, 근무강도 괜찮음, 연봉인상률은 작지만 실적 좋아 상여 잘 나옴"으로 상대적으로 긍정적
- **연봉**: 검색 스니펫 기준 대리 1년차 약 5,500만원, 대졸초임 CJ그룹 공채 기준 4,300~4,600만 — 다만 스니펫 출처가 혼재돼있어 신뢰도는 참고 수준
- **단점 반복 언급**: 과도한 보고 문화, 정치질/라인타기, 경쟁문화 과열, 구내식당 없음(CJ 계열 40% 할인 등 복지는 동일하게 적용)

## 기술 문화 — CJ올리브네트웍스 대비

- **oliveyoung.tech 기술블로그 실측**: 최근 발행 빈도 **월평균 약 3건**, 최근 글 예시 — "프레임워크에 구애받지 않는 통합 백오피스 구축하기"(07.14), "옴니채널 재고 정합성 인벤토리 파이프라인 구축기"(07.01), "에러로그 하나에 깨던 새벽에서 벗어나기까지 — 상품 모니터링 진화기"(06.30), "45분 배치→준실시간 Kafka 전환기"(04.22), "올영매장은 MSA 환경에서 흩어진 도메인 데이터를 어떻게 연동했을까"(03.18), "QA가 서버를 죽여본 이유 — 카오스 엔지니어링"(03.30) 등. **Kafka·MSA·카오스엔지니어링·AI 협업 프로세스 등 실제 최신 기술 스택을 다루는 글이 꾸준히 나옴** — CJ올리브네트웍스(2022.07 이후 방치)와 뚜렷이 대비됨
- CJ올리브네트웍스 재직자 평판 리서치(`cj-olive-networks-employer-research.md`)에서 확인된 "올네는 SI회사, 진짜 개발하려면 올영 가야 함"이라는 블라인드 발언과 정합 — **자사서비스 개발 조직 vs SI성 조직**이라는 구도가 기술블로그 활성도로도 뒷받침됨
- 다만 채용공고 자체에는 K8s/컨테이너 오케스트레이션이 필수요건으로 없음에도 기술블로그에는 MSA·Kafka 사례가 나온다는 점은, **포지션(예: 상품 메타)에 따라 실제 인프라 노출 정도가 다를 수 있다**는 뜻 — 지원 시 면접에서 "이 팀의 실제 배포/인프라 스택"을 직접 확인 권장

**추천**: **CJ올리브영 "백엔드 개발자(상품 메타)"가 갭이 가장 적고, 조직 평판·기술 문화 면에서도 CJ ONE보다 유리한 대안으로 판단됨.** CJ ONE은 마감이 확정(07.31)돼 있어 계획대로 진행하되, 상품 메타 포지션은 상시채용이라 서두를 필요는 없지만 **동시 지원을 검토할 가치가 충분함** — 같은 CJ 계열이라 이력서 기반 재사용 가능하고, 자소서 포인트만 "인증/멤버십" 대신 "상품 도메인" 중심으로 조정하면 됨. 단, 두 회사 동시 지원 시 이력서에 노출되는 지원 이력이 겹칠 수 있으니 인사 프로세스상 문제 없는지는 원티드/공고 페이지에서 별도 확인 권장(계열사 간 중복지원 제한 여부는 이번 조사에서 확인 안 됨).
