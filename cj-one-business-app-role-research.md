# CJ ONE 사업방향성·앱 기능·회원서비스 직무 범위 리서치 (2026.07.22)

CJ ONE 백엔드 개발자 자소서·면접 반영 목적. WebSearch/WebFetch 기반, 일부는 접속 실패로 미확인 — 각 섹션에 한계 명시.

## 1. CJ ONE 사업 방향성 — 최근 뉴스

- 2023.09 13년 만에 브랜드 리뉴얼("반짝이는 일상", 프리즘 BI) → **"멤버십 슈퍼앱"** 전환 선언, 이 기조가 2026년까지 이어짐
- 회원수: 2023.09 약 2,960만 → 2025.01 **3,100만 명 돌파**, 계속 순증
- **개방형 멤버십** 확장: CJ 계열(올리브영·뚜레쥬르·CGV·빕스·투썸플레이스) 넘어 **메가MGC커피, CU, HD현대오일뱅크, 이마트24, 야놀자(NOL)** 등 비계열 제휴 확대 — 2025.04 기준 약 60여 곳
- **핀테크 확장(신규 축)**: CJ PAY(자체 간편결제) 기반, 2025년 우리은행과 "**CJ PAY 우리통장**" 출시(금융위 혁신금융서비스 샌드박스 지정, 최대 3% 금리·2% 적립) — 멤버십 포인트를 예금상품과 결합
- AI/개인화: CJ올리브네트웍스가 마테크 사업으로 **Braze·Amplitude 공식 리셀링 파트너십**을 맺고 CJ ONE에 실제 적용 — 앱 행동 데이터 기반 개인화 메시징(포인트 안내, 혜택 추천). 타사(롯데쇼핑·신세계인터내셔날)에 컨설팅으로도 판매하는 사업모델 일부
- 유인상 대표 "2026년 매출·기업가치 1조 목표" 안에서 "CJ ONE 슈퍼앱 진화"가 최우선 과제. CJ ONE을 "플랫폼 사업구조"로 전환해 트래픽 기반 광고사업(**CJ ONE AD**)도 확대 방향 명시
- 포인트 유효기간 연장(공정위 지침, 2년→최대 5년) 이슈가 언급되나 CJ ONE 구체 시행 시점은 **미확인**

## 2. CJ ONE 앱 전체 기능

- 앱스토어 평점 4.5/5(약 7,800개), 라이프스타일 카테고리 48위(스냅샷, 변동 가능)
- 핵심 기능: 포인트 적립/사용, 쿠폰(신규가입·생일·VIP 전용), 기프트카드, 일일 미션(매일룰렛, 펀타운, **ONE워크**-걸음수 기반 포인트, 오늘운세), 위치기반 원더랜드, 매장찾기
- **주의**: 요청하신 "원스터" 캐릭터 정확 명칭은 이번 조사로 확정 못 함(ONE워크 관련 캐릭터로 추정) — 자소서/PT에 캐릭터명을 단정적으로 쓰기 전에 **본인이 앱을 직접 열어 재확인 필요**
- 인증/로그인: 앱 요구 권한에 생체인증(지문·얼굴) 정보 명시 → **생체인증 기반 간편 로그인 지원 확인**. 회원가입 시 별도 본인인증 단계 존재
- 최신 버전(스냅샷 기준 4.9.7)에서 홈 화면이 "멤버십 탭"/"라운지 탭"으로 개편됨 — 릴리즈노트에 인증/보안 관련 구체 문구는 확인 안 됨(일반적 "버그 수정" 문구)
- 반복 사용자 불만: UI/UX 급변경에 대한 불만, 실행 오류, 성능 저하. 포인트 적립 오류 관련 구체 리뷰는 미확인
- CJ온스타일 등 온라인몰은 "상품 수령 후 [포인트받기] 버튼을 눌러야 적립"되는 수동 확인 구조(FAQ 확인) — 반품·취소 대비 정산 정합성 설계로 추정, 동시에 사용자 불만(적립 누락 인지) 원인 가능성

## 3. "회원서비스" 직무 — 실제 담당 업무 범위 (★ 가장 중요한 발견 포함)

### ★ 기술스택 관련 중요 발견 — 반드시 인지할 것
헤드헌팅 플랫폼(비즈니스피플)에서 **"CJ ONE 백엔드 엔지니어" 공고 2건(BR250930A00183, BR251111A00211)을 확인했는데, 기술스택이 Kotlin/Ktor로 명시**되어 있음. 이게 지원 중인 recruit.cj.net 공고("CJ ONE 회원 서비스 백엔드 개발자", Java/Spring 명시)와 **완전히 동일한 포지션인지는 미확인**이지만, 같은 "CJ ONE" 서비스를 대상으로 한 공고라 CJ ONE 백엔드 조직 전체의 기술 방향성을 보여주는 신호로 볼 수 있음.

- 해당 공고 스택: Kotlin, Ktor(Core) / AWS+Terraform / Kubernetes(EKS)+Helm / PostgreSQL / Kafka+SQS / DynamoDB+Cassandra / Valkey(캐시) / Jenkins+ArgoCD
- 자격요건에 "회원 100만 이상의 대규모 자사 서비스 개발 경험" 명시 — recruit.cj.net 공고와 도메인이 겹칠 가능성 시사
- 서비스 소개 문구는 recruit.cj.net 공고와 거의 동일("CJ ONE은 올리브영·온스타일·뚜레쥬르 등에서 시작한 CJ그룹 통합 회원/멤버십포인트 시스템, 메가MGC커피·신세계면세점 등 제휴, 3천만+ 회원")
- **해석**: CJ ONE 백엔드 조직 내에 **Java/Spring 팀과 Kotlin/Ktor 팀이 공존**하거나, 조직이 스택을 전환 중일 가능성. 지원 중인 공고는 Java/Spring을 필수요건으로 명시하므로 당장 Kotlin이 요구되는 건 아니지만, **면접에서 "Kotlin 팀도 있던데 아시나요/괜찮나요" 류 질문이 나올 가능성을 염두에 둘 것**
- 비교: CJ올리브영(별도 법인) "코어플랫폼유닛" 백엔드는 Java/Kotlin + Spring + JPA/Hibernate 스택, 담당은 상품/회원/마이페이지/장바구니/결제 — CJ ONE과는 별개 조직·스택
