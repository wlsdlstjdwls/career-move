# CFT(인성검사) 대비 — CJ올리브네트웍스

`job-change-checklist.md` 섹션 2 대응, `cj-one-application-draft.md` 전형 단계 중 "CFT(인성검사)" 파트. 프로그래머스 플랫폼에서 온라인으로 진행되는 것으로 확인됨(1차 면접 통과 후 또는 코딩테스트와 같이 안내될 가능성 — 실제 일정은 서류 통과 후 공지 기준).

## 1. CFT가 뭔지 — 벼락치기가 안 통하는 이유

대기업 인적성 인성검사는 대부분 "같은 성향을 다른 문항으로 여러 번 물어서 답이 일관되는지" 보는 구조(사회적 바람직성 척도/라이 스케일 포함). 예를 들어 "나는 항상 규칙을 지킨다"와 "가끔 사소한 규칙은 어겨도 된다고 생각한다"를 둘 다 강하게 긍정하면 "일관성 낮음/좋게 보이려는 답변"으로 잡힘.

**그래서 전략은 "좋아 보이는 답"을 고르는 게 아니라, 본인 성향을 미리 몇 개 축으로 정리해두고 그 축에서 안 벗어나게 답하는 것.** 아래 2번이 그 축 정리.

## 2. 본인 성향 축 정리 (문항 답할 때 기준으로 삼을 것)

기존 자료(`interview-prep-notes.md`, `cj-one-application-draft.md`)에서 이미 드러난 성향 기준으로 정리. 실제 검사 볼 때 이 방향에서 벗어나지 않게 답하면 됨.

| 축 | 본인 성향 (근거) | CFT에서 나올 수 있는 문항 예시 |
|---|---|---|
| 조직 충성도 / 이직 성향 | 재입사 이력 있음 — "사람·문화는 만족, 처우·성장은 별개" 프레이밍 이미 확정(`interview-prep-notes.md` "재입사할 만큼 좋았다면서 왜 또 이직" 참고). **이 축이 재입사 이력 있는 지원자에게 가장 예민하게 체크될 가능성 높음** | "나는 조직에 대한 소속감이 강한 편이다" / "더 나은 조건이 있으면 언제든 이직할 수 있다" — 둘 다 극단으로 답하면 모순으로 보임. "소속감은 있지만 처우·성장은 별개로 판단한다"는 톤 유지 |
| 책임감/오너십 | Fitness In Service 기획~배포 전 과정 단독 수행, 기간계 고도화 AA 역할 — 실행 근거 뚜렷 | "맡은 일은 끝까지 책임진다" 계열 문항에 자연스럽게 긍정 가능(과장 아님) |
| 협업 스타일 | ExtJS 교육 직접 진행, 코어/공통 모듈 설계로 팀 기준 맞춤 — "주도해서 구조 만드는" 스타일이지 "지시받아 따르는" 스타일 아님 | "나는 팀에서 주도적으로 의견을 내는 편이다"에 긍정 / "다수 의견에 맞추는 걸 선호한다"에는 중립~약긍정 정도로, 극단적 긍정은 위 성향과 모순 |
| 문제 상황 대응 | 바우처 정책 변경 대응(오픈 직전 이슈, 전수조사+정규식 일괄교체) — 압박 상황에서 체계적으로 접근하는 사례 | "예상치 못한 문제가 생기면 침착하게 대응한다" 계열에 근거 있는 긍정 가능 |
| 안정성/규칙 준수 | 회사코드 반출 금지 등 원칙 지키는 태도 확인됨(`offboarding-guide.md`) | "규칙과 절차를 중요하게 생각한다"에 긍정해도 무리 없음 |

## 3. 준비 체크리스트

- [ ] 위 표 5개 축을 본인이 다시 읽고 "이게 진짜 내 성향 맞나" 스스로 점검 (내가 정리한 건 기존 자료 기반 추정 — 실제 본인이 아니라고 느끼면 그쪽이 맞는 것)
- [ ] 검사 직전엔 "정답 맞히기"로 접근하지 말고 위 축을 기준으로 편하게 답하기 — 문항 수 많은 검사에서 매번 계산하며 답하면 오히려 일관성이 깨짐
- [ ] 재입사 이력 관련 문항(소속감/이직 성향 계열)이 나오면 특히 유의 — 자소서·면접 답변과 결이 다르게 나오면 서류는 통과했는데 CFT에서 삐끗하는 경우가 생길 수 있음
- [ ] 검사 전날 컨디션 관리(충분한 수면) — 극단적 응답 패턴은 피로/서두름에서도 나옴

## 4. CFT 이후 — 면접에서 인성검사 결과 관련 질문 나올 경우

CJ 면접 후기(`cj-one-application-draft.md` "4. 면접 후기 리서치" 참고)에 따르면 기술보다 경험·인성 질문 비중이 높다고 함 — CFT에서 나온 성향과 면접 답변이 다르면 면접관이 바로 캐치할 가능성 있음. 위 5개 축을 면접 답변 준비할 때도 같은 톤으로 유지할 것.
