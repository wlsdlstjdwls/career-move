## 단계 표기 기준 (예시)
- 서류 → 코딩테스트 → 1차 면접 → 2차 면접(임원) → 처우 협의 → 최종
- 회사마다 프로세스가 다르므로 실제 공고 확인 후 단계명 조정

## 사용 방법
1. `target-companies.md`에서 실제 지원할 회사가 정해지면 이 표에 행 추가
2. 공고 확인 즉시 "공고일" 기재, 지원 완료 시 "지원일"·"현재 단계" 갱신
3. 탈락/합격 등 결과가 나오면 "결과" 컬럼에 기록 (다음 지원 전략에 참고)
