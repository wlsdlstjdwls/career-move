# CJ올리브네트웍스 자소서 작성 전략 — 유튜브 강의 리서치 (2026.07.22)

출처: [해커스잡 · 조은희 강사 — CJ올리브네트웍스 채용 전형/자소서 강의](https://www.youtube.com/watch?v=ZI2rScVYRcg) (자동생성 자막 기반, 도입~비전 활용 구간)

CJ ONE 자소서 최종본(`cj-dashboard.html` "자기소개 실제 문항 최종본" 참고) 제출 전 자가 점검용. 영상은 CJ올리브네트웍스 공통 자소서 전략 강의라 CJ ONE 직무에도 그대로 적용됨.

**⚠️ 참고용 한정**: 이 영상은 신입 취준생 대상 강의임(2026.07.22 확인). 전형 순서·표절검수·글자수 미달 리스크 등 일반 원칙은 경력직에도 유효하나, **키워드 나열·미션 문구 삽입 권장은 신입 자소서 작법**에 가까움. 경력직은 실제 경험·성과로 지원동기를 증명하는 게 우선이므로, 아래 5번(미션/키워드 활용) 항목은 강제 적용하지 말고 참고만 할 것.

## 1. 채용 전형 순서 (영상 명시)
서류전형 → 테스트전형(인적성+직무테스트) → 1차면접 → 2차면접. CJ 인적성 난이도가 어려운 편이라 서류 기간이 긴 이유 중 하나로 지목됨. `cft-personality-test-prep.md`와 별개로 **인적성 자체 학습**도 병행 필요.

## 2. 자소서 비중과 진짜 이유
- CJ ON은 자소서를 **키워드 기준으로 필터링** — 스펙 비중(약 80%)이 더 크지만 자소서도 탈락 요인이 됨.
- 진짜 중요한 이유는 **면접에서 자소서 기반 꼬리질문 + 역량자료(인적성)와 비교분석**이 나오기 때문. 자소서에 없는 내용을 면접에서 지어내면 인적성-면접 정합성이 깨짐 → 신뢰도 하락.
- 4번 항목류에서 "CJ 상품·서비스에 실제로 관심 있는지" 확인하는 문항이 반복 출제되는 트렌드 있음.

## 3. 탈락 사유 (네거티브 체크)
- **글자 수 미달 = 불성실 기재로 탈락 가능**. 예: 1000자 항목에 400~500자만 채우면 위험. → CJ ONE 문항은 최대 10,000자라 미달 리스크는 낮지만, **상한의 상당 부분을 채우는 밀도**는 챙길 것.
- 표절 검수: 카피킬러·네이버·사람인 등으로 검수됨. 오타율 높으면 감점.

## 4. ChatGPT/AI 활용 원칙 — CJ ONE 자소서 적용 체크
- 문항 전체를 GPT로 돌리면 "티가 난다" → 비추천.
- **본질적 내용(경험·성과·수치)은 본인이 직접 작성**해야 함 — 면접에서 그대로 재현 가능해야 하기 때문.
- GPT는 **가독성 개선, 문장 길이 조절, 자연스러운 연결, 키워드 반영**에만 사용.
- → 이미 `cj-dashboard.html`의 humanize-korean 윤문 프로세스(등급 A, 변경률 20%, 사실관계 100% 보존)가 이 원칙과 정확히 일치함. 추가 조치 불필요, 다만 **면접 전 자소서 내용을 소리 내어 재현하는 연습**은 별도로 필요.

## 5. 미션/키워드 활용 — 문항 2(지원동기) 재점검 포인트
- CJ올리브네트웍스 미션 문구("OnlyOne 기술과 서비스로 고객의 성공을 견인하고 더 나은 세상에 기여")를 **포부(마무리 문단)**에 변형 삽입 권장.
- 활용 키워드 예시: 고객 중심, 사회 가치 증대, 전문성 강화, 혁신, 차별화, 상생, 최고 — **전부 넣지 말고 본인 성향에 맞는 2~3개로 컨셉을 정해 일관되게 사용**.
- 현재 문항2 초안은 "CJ ONE 슈퍼앱 진화"·"하고잡이"(`cj-youtube-hagojabi-dxax-research.md` 리서치 반영)는 들어가 있으나, **"OnlyOne" 미션 문구는 아직 미반영** — 포부 문단에 추가 검토 대상.

## 액션 아이템 (제출 전 체크리스트)
- [ ] 문항2 마무리(포부) 문단에 "OnlyOne" 미션 키워드 자연스럽게 1회 삽입 검토
- [ ] 자소서 전체에서 "고객 중심" 계열 컨셉 1~2개로 일관되게 정리됐는지 재확인 (현재 초안은 "고객 자산·신뢰" 축으로 이미 일관성 있음 — 유지 권장)
- [ ] 인적성 검사 난이도 대비 — CFT 성향 정리(`cft-personality-test-prep.md`)와 별개로 **인적성 자체 문제풀이 연습** 일정에 추가
- [ ] 면접 대비: 자소서에 쓴 프로젝트(더현대·투홈 통합, PC몰 리뉴얼, 기간계 고도화 등) 전부 소리 내어 설명 연습 — 자소서-인적성-면접 정합성 확보 목적
- [ ] 제출 전 자체 표절검사(카피킬러 등) 1회 실행 권장
