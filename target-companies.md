## 타겟 회사 기술 블로그 리서치 (2026.07.20)

면접 시 "저희 팀/회사에 관심 있는 이유" 답변 근거 확보용. 우선순위 4곳(application-tracker.md 기준) 확인 결과 — **대기업 그룹 IT 계열사는 대체로 공개 기술 블로그를 활발히 운영하지 않는다**는 게 확인됨. 이건 예상 밖 결과라 그대로 기록.

- **CJ올리브네트웍스(CJ ONE 백엔드, 최우선)**: 자체 `cjolivenetworks.co.kr/news/blog`는 사실상 방치 상태(최근 글이 2022.07). 대신 같은 CJ 계열의 **올리브영 테크블로그(oliveyoung.tech)**가 매우 활발함 — 최근 글: "옴니채널 재고 정합성 대응 인벤토리 파이프라인"(Kafka·멀티스레드, 2026.07), "다수 도메인 데이터 Kafka 통합 — 45분 배치→준실시간 전환"(Kafka+Redis 하이브리드, 2026.04), "MSA 환경 도메인 데이터 연동"(Redis Cache-aside, Event-Driven, 2026.03), "배송최적화 시스템 구축기"(Rule Engine, 2026.03). **주의**: CJ올리브영과 CJ올리브네트웍스는 CJ 계열이지만 법인이 다름 — 면접에서 "이 글 봤다"고 할 때 "같은 CJ 생태계의 커머스 기술 사례로 참고했다"는 식으로 정확히 표현할 것, 마치 지원 회사 자체 블로그인 것처럼 말하면 안 됨.
- **현대이지웰**: 공개 기술 블로그 확인 안 됨. 대안: 복지몰 개편 관련 보도자료(AWS 전환, AI 추천, 재입고 알림 등)를 관심사 근거로 활용하거나, 면접 시 "더현대·투홈 NCP 구축" 본인 프로젝트 경험(포인트/고객자산 도메인)으로 연결해 설명하는 쪽이 더 실질적.
- **신세계아이앤씨**: 공개 기술 블로그 확인 안 됨. SSG.COM(그룹 커머스 계열사)도 "SW개발 내재화" 언급은 있으나 공개 기술 블로그는 확인 안 됨. 자체 개발 플랫폼(Java 기반 프레임워크, 공통 컴포넌트/모듈 라이브러리)을 갖고 있다는 점 정도가 공식 자료로 확인 가능한 정보.
- **롯데이노베이트(구 롯데정보통신)**: 공개 기술 블로그 확인 안 됨. 사명 변경(위 표 참고) 이후 신사업(메타버스·전기차 충전·자율주행) 확장 방향성 정도가 뉴스로 확인 가능.

**결론**: 대기업 그룹 IT계열사 지원 시 "기술 블로그 읽고 왔다"는 어필이 통하는 곳은 사실상 CJ뿐(그것도 자매회사 블로그). 나머지는 블라인드/잡플래닛 면접 후기나 회사 뉴스룸의 사업 방향성 기사로 관심사 근거를 대체하는 게 현실적 — 이 부분은 `job-change-checklist.md` 6번(면접 후기 확인) 항목과 연결해서 준비할 것.

## 이름 확인 완료 — "현대이지웨어" → 현대이지웰

앞서 "현대이지웨어"라는 이름이 검색에 잡히지 않아 혼선이 있었음(2026.07.20). 본인 확인 결과 **현대이지웰(현대EZwel)**이 맞는 것으로 확정 — 2021년 현대백화점그룹에 편입된 복지솔루션 회사. 사업 영역이 기업복지·식권·BPO라 얼핏 이커머스와 거리가 있어 보이지만, 실제로는 **복지몰 자체가 AWS 기반의 쇼핑몰형 이커머스 플랫폼**(상품/주문/포인트결제/AI추천)이라 도메인 연결성이 낮지 않음 — 위 표에 반영 완료. 문서 전체(이 파일·`application-tracker.md`)에서 회사명을 현대이지웰로 정정함.

## 타겟 회사 면접 후기 리서치 — 신세계아이앤씨 / 롯데이노베이트 (2026.07.20)

**신세계아이앤씨** (잡플래닛 기준 면접난이도 2.9, 긍정 34%/부정 13%, 합격률 62%)
- 블라인드 면접 진행 — 학교/전공 언급 안 하도록 요청받음
- 실무진 면접·Dream Stage(임원급 추정) 면접 모두 **다대일(면접관 여러 명 : 지원자 1명)** 방식
- 실제 후기 사례: "백엔드 기술 부족"에 대한 지적성 질문 + "향후 1년 내 백엔드 공부 계획"을 물어본 케이스 있음 — 부족한 부분을 인정하되 구체적 학습 계획으로 답하는 게 중요하다는 시사점(CJ ONE의 K8s 갭 대응과 같은 논리, `interview-prep-notes.md` 참고). AWS 사용 경험도 질문받았는데, 실제 없으면 솔직히 답하라는 후기 팁 — 본인은 AWS EC2 경험(Fitness In Service) 있으니 유리

**롯데이노베이트(구 롯데정보통신)** (잡플래닛 기준 면접난이도 2.9, 긍정 48%, 합격률 55%)
- 면접 구성: **역량면접 + 인성면접(임원면접)** 2단계, 난이도는 높지 않은 편이라는 후기 다수
- 공백기·나이에 대한 특별 질문은 없었다는 후기 — 재입사 이력에 대한 부담을 크게 가질 필요는 없어 보임(그래도 `interview-prep-notes.md` 답변은 준비해둘 것)
- **블라인드 커뮤니티에서 나온 우려 사항(참고만 할 것, 지원 여부 판단에 참고)**: 네카라쿠배 등 서비스 기업 대비 연봉·성과급 상승률이 낮고 스톡옵션 기대 어렵다는 평, 기술 스택이 "최신을 따라가려 하지만 실제로는 기초적인 수준"이라는 평, 실력 있는 개발자 이탈이 많아 배울 사람이 적다는 평도 있음 — `offer-comparison-framework.md`의 "기술 스택 성장성"·"조직 안정성" 비교축에서 낮은 점수를 줄 만한 근거. 최종 판단은 커피챗(`networking-templates.md`)으로 직접 확인 권장
